# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

## [1.0.0] - 2026-03-28

### Adicionado
- Cadastro de medicamentos com nome, paciente, horário, frequência, estoque e observações
- Listagem completa de medicamentos cadastrados
- Busca de medicamentos por nome do paciente
- Atualização de estoque de medicamentos
- Remoção de medicamentos com confirmação
- Alertas automáticos para estoque abaixo de 5 unidades
- Persistência ded dados em arquivo JSON
- Interface CLI interativa com menu numerado
- Testes automatizados com pytest
- Lint configurado com flake8
- Pipeline de CI com GitHub Actions
